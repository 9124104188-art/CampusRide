🚗 CampusRide

CampusRide is a full-stack MERN (MongoDB, Express.js, React, Node.js) web application that connects college students for ride sharing. It allows students to create rides, join available rides, manage their own trips, and update ride status through a secure authentication system.

✨ Features
🔐 User Authentication using JWT
👤 Student Registration & Login
🚘 Create Ride
🔍 Browse Available Rides
✅ Join and Leave Rides
✏️ Edit Ride Details
🗑️ Delete Created Rides
🚦 Update Ride Status (Available, Started, Completed, Cancelled)
🔎 Search and Filter Rides
📱 Responsive User Interface
💾 MongoDB Database Integration

🎯 Objective

CampusRide aims to simplify transportation for college students by providing a secure platform to share rides, reduce travel costs, and encourage eco-friendly commuting.
